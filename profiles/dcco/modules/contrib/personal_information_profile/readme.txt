--------------------------------------------------------------------------------
                              Personal Information Profile Feature
--------------------------------------------------------------------------------

Maintainer: 
 * Tim Osborn (yetiboy), tim@tcographicdesign.com
 

Allows users to input personal information after registration

Installation
-------------

 * Copy the whole personal_information_profile directory to your modules directory and
   activate the module.


Usage
-----
   
 * Users may input or edit their personal information by going to http://site_name/profile-personal-information
 * View of users profile information is available for all users at http://site_name/user-personal-information/user_name
* View: User Personal Information block shows 10 random profiles at a time with links to individual user pages and a page showing all users
* Links to all user profiles can be accessed at http://site_name/all-user-profiles
